# MovieCatalogue

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 13.1.1.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Running back-end server

Run `npm install -g json-server` to globally install the json server. After the installation go into the project folder and run `json-server --watch db.json` this will initialize a server at the url `http://localhost:3000/` after the initialization it will be possible to make http requests.
